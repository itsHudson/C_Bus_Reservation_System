# BusReservationSystem

# Tool Using
Code::Blocks
- Version: MinGW - Microsoft Windows (64 bit, default)
- URL (Binary Releases): https://www.codeblocks.org/downloads/binaries/
- URL (Mingw-setup.exe): https://sourceforge.net/projects/codeblocks/

Code::Blocks MinGW: 
- Version: MinGW - Minimalist GNU for Windows
- URL: https://sourceforge.net/projects/mingw/
- In MinGW Installation Manager, tick the following packages:
  - `mingw32-base`
  - `mingw32-gcc-g++`
  - `msys-base`

1. On the Windows search bar, type: Environment variable
   Press Enter.

2. Click the **Environment Variables…** button.

3. Under **System variables**, select **Path**, then click **Edit**.

4. Click **New** and add the following directory: C:\MinGW\bin
